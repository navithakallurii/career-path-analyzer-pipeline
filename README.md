# Career Path Analyzer Pipeline

## Overview
This project simulates a data pipeline that helps identify career paths for students based on their skills.

## Features
- Data cleaning (handling missing values)
- Skill-based career mapping
- Output generation for further AI/ML use

## Tech Stack
- Python (Pandas)

## Use Case
This pipeline can be extended into an AI system that recommends career paths based on user skills and interests.
